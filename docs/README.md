# Appliance Service App — Documentation

This folder contains the living documentation that drives development of the Appliance Service App, built to showcase knowledge and lessons learned from the AI Bootcamp.

## Structure

| Folder           | Purpose                                                      |
| ---------------- | ------------------------------------------------------------ |
| `architecture/`  | System design, component diagrams, data flow, tech decisions |
| `guidance/`      | Coding standards, conventions, and AI-assisted workflow tips  |
| `specs/`         | Detailed technical specifications for modules and APIs       |
| `features/`      | Feature descriptions, acceptance criteria, and user stories  |

## How to Use

1. **Before building** — check `features/` for the user story and acceptance criteria.
2. **While designing** — reference `architecture/` for component structure and data flow.
3. **While coding** — follow `guidance/` for conventions and best practices.
4. **While implementing** — use `specs/` for detailed technical requirements.
